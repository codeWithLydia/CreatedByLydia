Portfolio project

Doel van het project

Een portfoliopagina vormgeven en bouwen die laat zien wat ik tot nu toe geleerd heb. De pagina moet zoveel mogelijk aan de toegankelijkheidseisen voldoen.

Proces

In Figma heb ik een wireframe gemaakt voor mobile en desktop. De foto's op de website heb ik allemaal zelf gemaakt, hiervoor moest ik mijn fotobibliotheek uitpluizen. De website is gebouwd in html en CSS met een beetje JavaScript.

Eindresultaat

Een portfolio website waar ik blij mee ben, die bij mij past, laat zien waar ik voor sta en ga en die laat zien wat ik tot nu toe geleerd heb. Uiteraard is dit een ongoing process en is de website nooit helemaal af.
